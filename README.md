# Portfolio Business Analyst

## Giới thiệu
Đây là website portfolio cá nhân được xây dựng nhằm phục vụ cho môn học và định hướng nghề nghiệp Business Analyst.

---

## Mục tiêu
- Xây dựng portfolio cá nhân chuyên nghiệp
- Trình bày dự án và kỹ năng chuyên môn
- Làm quen với Git, GitHub và deploy website

---

## Công nghệ sử dụng
- HTML, CSS
- Git & GitHub
- GitHub Pages

---

## Nội dung website
Website bao gồm các phần:

### 1. Giới thiệu
- Thông tin cá nhân
- Định hướng nghề nghiệp BA

### 2. Dự án
- Quản lý cửa hàng 24H (Web app)
- Quản lý khóa luận (Web app)
- Quản lý thư viện (Winform)
- Bao gồm:
  - Mô tả
  - Hình ảnh minh họa
  - Vai trò trong từng dự án

### 3. Kỹ năng
- Business Analysis
- Technical Background
- Process & Management

---

## Quá trình thực hiện

### Bước 1: Thiết kế giao diện
- Tham khảo mẫu CV
- Thiết kế layout bằng HTML + CSS

### Bước 2: Xây dựng nội dung
- Viết phần giới thiệu
- Chuẩn bị hình ảnh dự án

### Bước 3: Sử dụng Git
- Khởi tạo repository local
- Commit và push code lên GitHub

### Bước 4: Deploy
- Sử dụng GitHub Pages để deploy website
